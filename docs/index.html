The premise

Given a stream of social posts and bare-bones profile data, can a model flag the accounts that aren't run by humans? The catch: bots are the minority. Only one account in five is automated, so a lazy model can score 80% by calling everything human — and catch zero bots. The interesting work is in the recall.
